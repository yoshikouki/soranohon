# 挿絵計画生成プロンプトテンプレート

このファイルには、挿絵計画を生成するための Claude 用プロンプトテンプレートが含まれています。

## 登場人物設定プロンプト

```
あなたは日本の絵本用キャラクターデザイナーです。以下の物語の登場人物「CHARACTER_NAME」の設定画を生成するためのプロンプトを作成してください。

物語の情報：
CHARACTER_DESCRIPTION

目標：
- 子供向けの日本の絵本に適したキャラクターデザイン
- 物語の時代背景や環境に合った姿
- 性格や特徴が視覚的に表現されたデザイン

以下の要素を考慮してプロンプトを作成してください：
1. キャラクターの外見（服装、髪型、表情など）
2. キャラクターの持ち物や特徴的な小道具
3. キャラクターの姿勢や体の動き
4. 背景の要素（必要であれば）
5. 画風（日本の絵本風、優しい色彩など）

回答形式：
- プロンプトは日本語で作成し、画像生成AIに入力できる形式にしてください
- 100-150単語で簡潔に作成してください
- キャラクターの個性が伝わるような具体的な表現を使ってください
```

## 挿絵シーンプロンプト

```
あなたは日本の絵本用イラストレーターです。以下の物語のシーンの挿絵を生成するためのプロンプトを作成してください。

シーン情報：
SCENE_DESCRIPTION

目標：
- 子供向けの日本の絵本に適した挿絵
- 物語の雰囲気や感情を伝える絵
- 登場人物と環境の関係性が明確に表現された構図

以下の要素を考慮してプロンプトを作成してください：
1. 登場人物の配置と表情
2. 背景の描写（場所、時間帯、天候など）
3. 色彩とトーン（明るい/暗い、鮮やか/くすんだなど）
4. 視点と構図（俯瞰/アイレベル、クローズアップ/ワイドショットなど）
5. 重要な視覚的要素や小道具

登場人物：CHARACTER_LIST
場所：SETTING_DESCRIPTION
雰囲気：MOOD_DESCRIPTION
視覚的要素：VISUAL_ELEMENTS

回答形式：
- プロンプトは日本語で作成し、画像生成AIに入力できる形式にしてください
- 150-200単語で簡潔に作成してください
- シーンの感情的な雰囲気や物語上の重要性を伝える表現を使ってください
```

## 完全な挿絵計画プロンプト

```
あなたは絵本の挿絵計画を作成する専門家です。提供されたMDXファイルの内容を分析し、物語に適した挿絵計画を作成してください。

指示：
1. MDXファイルの内容を読み、物語の流れと主要な登場人物を特定してください。
2. 10-12枚の挿絵を配置するのに最適な場所を判断してください。
3. 各挿絵について、以下の要素を含む詳細な計画を作成してください：
   - 挿絵のタイトル
   - 物語内での場所（行番号）
   - シーンの簡潔な説明
   - 登場人物と彼らの表情、姿勢、動き
   - 背景の描写
   - 全体的な雰囲気や色調
   - 強調すべき視覚的要素
4. 各主要登場人物について、キャラクター設定画も計画してください。
5. 各挿絵とキャラクター設定画に対して、AI画像生成用のプロンプト案を提案してください。
```
